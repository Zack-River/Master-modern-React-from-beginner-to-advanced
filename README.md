#### useState
- useState is a hook that is used to manage the state of a component
- we can destructure the value and the setter function
- the value is the current state
- the setter function is used to update the state
- the setter function is asynchronous
- the setter function is not guaranteed to update the state immediately
- by default the state is immutable and value is ""
- we can pass a function to the setter function and it will be called with the previous state

#### jsx as a return value
- jsx is a syntax extension of javascript
- it is not a new language
- it is a syntactic sugar for react.createElement()
- it is a way to write html in javascript
- it is a way to write javascript in html
- it is a way to write jsx in jsx
- we can return elements a normal html js and we also can use the state vars or normal vars inside with curly braces {}

#### useEffect
- useEffect is a hook that is used to manage the side effects of a component
- it is a function that is called after the component is rendered
- it is a function that is called before the component is rendered
- it is a function that is called after the component is updated
- it is a function that is called before the component is updated
- it is a function that is called before the component is unmounted
- it is a function that is called after the component is unmounted
